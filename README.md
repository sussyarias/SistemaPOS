# SistemaPOS
Sistema POS (Point of Sale) desarrollado en Python con gestión de inventario, facturación térmica y soporte multi-caja vía PostgreSQL.

El objetivo de este proyecto es construir un sistema de gestión de inventario sencillo que ajuste el stock automáticamente al momento de facturar, con soporte para impresión térmica directa.

## Features actuales
* Impresión térmica directa (impresora AON PR-100)
* Conexión y modelos en PostgreSQL
* Script de creación de tablas

## Features planeados
* Log-in con múltiples usuarios y roles
* Inventario con control de stock
* Historial de facturas
* Reportes de ventas por día

## Tecnologías
* Python
* Flask
* PostgreSQL
* HTML + Tailwind CSS
* win32print
